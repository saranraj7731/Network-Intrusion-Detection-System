 Network Intrusion Detection System

 Objective

To implement a Network Intrusion Detection System (NIDS) using Suricata on Kali Linux for monitoring network traffic and detecting suspicious activities.

Tools Used

* Kali Linux
* Suricata

Project Steps

1. Installed and configured Suricata.
2. Verified network interfaces.
3. Configured custom detection rules.
4. Monitored network traffic.
5. Generated alerts and logs.
6. Analyzed intrusion detection results.

Results

* Successfully deployed Suricata IDS.
* Detected and logged suspicious network activity.
* Generated alerts through custom rules.

 Conclusion

This project demonstrates how intrusion detection systems can be used to monitor network traffic, detect threats, and improve overall security posture.
